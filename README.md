# 👁️ OcuSense: Retinal Lesion Detection System

OcuSense is a specialized medical imaging decision-support tool built with Python, Streamlit, and YOLO. It is designed to assist in the early screening of Diabetic Retinopathy by detecting and categorizing four primary types of retinal lesions from fundus photographs.

🚀 Features
Multi-Pipeline Preprocessing: Choose between Baseline and three advanced enhancement strategies (A, B, and C) to improve lesion visibility.

Intelligent Validation: Automatic Red-to-Blue channel ratio checking to ensure uploaded images are valid fundus photos.

Real-time Inference: High-speed detection using YOLO models with adjustable confidence thresholds.

Detailed Analytics: Visual lesion mapping on the original image paired with quantitative summary tables and confidence scores.

Clinical Context: Built-in educational resources regarding red and white retinal lesions.
